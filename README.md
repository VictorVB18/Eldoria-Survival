# Eldoria: The Lost Realm 🌲🐉

An immersive 3D survival RPG set in the mystical realm of Eldoria. Explore lush forests, survive ancient guardians, and witness the majesty of dragons in this high-fidelity browser experience.

![Eldoria Gameplay](public/menu_bg.png)

## ✨ Features
*   🎬 **Cinematic Intro**: A movie-style storytelling sequence for all new characters.
*   🌍 **Procedural World**: Explore diverse biomes including Magic Forests, Golden Plains, and Crystal Highlands.
*   🐉 **Ancient Creatures**: Encounter roaming giants and flying dragons.
*   💾 **Save System**: 4 persistent save slots to track your progress.
*   🎭 **Character Customization**: Personalize your adventurer's appearance.
*   🌅 **Dynamic World**: Real-time lighting and atmosphere transitions.

## 🎮 Controls
*   **WASD**: Movement
*   **SPACE**: Jump
*   **SHIFT**: Sprint
*   **MOUSE**: Look around
*   **LEFT CLICK**: Attack / Interact
*   **E**: Inventory / Crafting
*   **ESC**: Pause Menu

## 🚀 How to Play Locally
1.  Clone the repository:
    ```bash
    git clone https://github.com/YourUsername/Eldoria-Survival.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open `http://localhost:5173` in your browser.

## 🛠️ Built With
*   [Three.js](https://threejs.org/) - 3D Engine
*   [Vite](https://vitejs.dev/) - Frontend Tooling
*   [Simplex-Noise](https://github.com/jwagner/simplex-noise.js) - Procedural Generation
*   [Cannon-es](https://github.com/pmndrs/cannon-es) - Physics Engine
